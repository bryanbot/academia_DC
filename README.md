# 🚀 Proyectos y Ejercicios Incluidos

| Script | Descripción | Tópicos Cubiertos |
| :----------- | :------------: | ------------: |
| generateFileE1.py | Script interactivo para generar un archivo CSV de transacciones financieras simuladas.
